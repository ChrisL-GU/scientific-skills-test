================================================================================
INTERPRETATION GUIDE
================================================================================

Volcano Plot:
  ‚Üí Y-axis: Statistical significance (-log10 p-value)
  ‚Üí X-axis: Effect size (log2 fold change)
  ‚Üí Red dots: Significant changes (padj<0.05, |log2FC|>threshold)

Network Hub Proteins:
  ‚Üí Degree: Number of interactions (higher = more connected)
  ‚Üí Betweenness: Bridge between modules (higher = more central)
  ‚Üí Genes/proteins with high degree are key drivers

Model Performance:
  ‚Üí AUC > 0.9: Excellent discrimination
  ‚Üí AUC > 0.7: Good discrimination
  ‚Üí AUC ‚âà 0.5: Random classifier
  ‚Üí Confusion matrix: Shows True/False positive/negative rates

Pathway Enrichment:
  ‚Üí Number of proteins in pathway: Pathway activity level
  ‚Üí Mean log2FC: Direction of pathway regulation
  ‚Üí Network size: Number of inter-pathway connections

================================================================================
TROUBLESHOOTING
================================================================================

Q: ModuleNotFoundError: No module named 'pydeseq2'
A: Run: pip install -r requirements.txt

Q: FileNotFoundError: data/raw/rnaseq_counts.csv
A: Script generates example data automatically
   Or: Create data files matching DATA_FORMAT.md

Q: Results directory empty
A: Run complete pipeline: python scripts/00_run_full_pipeline.py

Q: ImportError: 'numpy' has no attribute 'float'
A: Update packages: pip install --upgrade -r requirements.txt

Q: Memory error with large datasets
A: Reduce number of features or samples
   Use subset of most variable features

Q: API rate limits exceeded
A: Example uses mock data (no real API calls)
   Production: Add rate limiting with time.sleep()

See README.md Troubleshooting section for more help.
